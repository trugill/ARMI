REQUIRED FILES
--------------
maxent.jar   - The MaxEnt species distribution modeling tool

DOWNLOAD
--------
Get MaxEnt from:
  https://biodiversityinformatics.amnh.org/open_source/maxent/

Place maxent.jar directly in this folder.

JAVA REQUIREMENT
----------------
MaxEnt requires Java 1.8 or later.
Verify with: java -version
